# In-Memory Cache Simulation

A Spring Boot project built to understand caching concepts through a Weather API simulation. This project demonstrates cache hits, cache misses, cache consistency, and the use of Spring Cache annotations such as `@Cacheable`, `@CachePut`, and `@CacheEvict` with an in-memory cache implementation.

The goal of this project is to explore caching behavior and gain hands-on experience with common caching patterns and performance optimization techniques in backend applications.
